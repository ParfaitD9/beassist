Pour ce projet vous aurez besoin de Python installé sur votre ordinateur
et de la bibliothèque wkhtml disponible à https://wkhtmltopdf.org/downloads.html

* Utilisateurs Windows
Vous devrez télécharger la bibliothèque et ajouter à vos variable d'environnement

*Aide
Lancez la commande python manage.py --help pour obtenir une brève aide
